O HTML

O HTML sigla para Hyper Text Markup Languague ou Linguagem de Marcação de Hipertexto não é propriamente dito uma linguagem de programação.

Como o prórpio nome sugere ele é considerado uma linguagem de marcação. Onde são inserido os comando (chamados de Tags) e com isso a página vai ganhando forma.

Basicamente a construção é dividida em duas parte: a cabeça e o corpo.

  1. Head (cabeça)

    2. Aqui são inserido todos os comandos que devem ser executados antes do usuário acessar a página.

  2. Body (corpo)

    3. Aqui são inseridos todos os comandos que são exibidos durante a visualização da página pelo usuário.

O VS CODE

Assim como em qualquer outra linguagem pode-se escrever uma página em HTML utilizando qualquer editor de texto, inclusive uma folha de papel.

Porém, facilita muito o trabalho quando utilizamos recursos específicos, que podem compilar nosso código. Assim, temos vizualizações e encontramos possíveis erros durante a execução do trabalho.

Nesse contexto, o VS CODE pode ser um ótimo aliado.

O VS CODEé uma ferramenta gartuita que pode ser utilizada para o desenvolvimento de diversas linguagens de programação. Inclusive o HTML.

Então, se você está interessado em começãr a aprender HTML, não perca tempo e instale e use o VS CODE na sua máquina.

As TAGs

A criação de um site utilizando HTML se dá basicamente utilizando Tags.

As Tags são palavras que servem como comandos, sua utilização define o que vai acontecer na página quando ela for exibida.

As Tags são sempre utilizadas entre os sinais de maior e menor. Como por exemplo:

< abc > - utilizada para abrir um comando.

< /abc > - utilizada para fechar o mesmo comando.

Assim tudo o que estiver entre os dois comandos responderão ao comando da TAG utilizada. Observação: abc não é uma Tag e foi utilizado apenas como exemplo didático.

Algumas Tags muito utilizadas para textos são:

  • h1, h2, h3, ... h6 - para títulos e subtítlulos.
  • p - para parágrafos.
  • strong - para negrito.
  • ol e ul - para listas numeradas e não numeradas
  • small - para letras pequenas.
